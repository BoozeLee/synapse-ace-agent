Synapse-Ace Autonomous Agent

OOBE Protocol × Ace Data Cloud Bounty Submission

What is this?

A fully autonomous TypeScript agent that discovers tools, executes multi-service AI workflows, and settles payments on-chain via x402 — all without human intervention.

🧠

SAP Registration

One-time on-chain identity creation on Synapse Agent Protocol mainnet.

🔍

Tool Discovery

Autonomously finds and catalogs available AI tools via SAP discovery.

⚡

Multi-Service Workflows

Orchestrates OpenAI chat, Flux image generation, and SERP search in a single task.

💳

x402 Payments

Automatic per-call settlement using AceDataCloud's x402 facilitator.

🛡️

Sentinel Validation

Integrates Synapse Sentinel for attestations and quality scoring.

💰

Escrow Management

SAP escrow account with automatic top-up and batch settlement.

Demo Walkthrough

Simulation mode (no real transactions). To run with real keys, see QUICKSTART.md.

Get Started

  1. Clone the repository: git clone https://github.com/BoozeLee/synapse-ace-agent.git
  2. Install dependencies: npm install
  3. Copy .env.example to .env and add your API keys.
  4. Register agent: npm run register
  5. Run demo: npm run demo
